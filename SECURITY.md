# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this MCP server, please report it responsibly.

**Email:** xltnapps@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 5 business days
- **Fix or mitigation:** as soon as reasonably possible, depending on severity

## Scope

This policy covers:
- The hosted MCP server at `mcp.octotrip.app/rental-cars`

Out of scope:
- Third-party provider APIs that this server queries
- Third-party booking sites linked from results

## Supported Versions

Only the latest released version is supported with security updates.

## Disclosure

We will coordinate disclosure with the reporter. We aim to release fixes before public disclosure.
